## 클래스

- 클래스 자체는 자바에서 많이 사용해서 익숙한데 파이썬의 클래스는 매개변수 같은 부분이 좀 특이하다고 생각됐다.

```python
class 클래스명:
	def __init__(self, 인수, ...):
		...
	def 메소드명(self, 인수, ...):
		...
```

- `__init__`: 자바의 생성자. 특별한 메소드로 클래스를 초기화하는 역할로 클래스의 인스턴스가 만들어질 때 한 번만 불리는 메소드.
- `self`: 파이썬에서는 메소드의 첫 번째 인수로 자신(자신의 인스턴스)를 나타내는 self를 명시적으로 씀. ⇒ 이게 생소한 느낌…

- 실습코드

```python
class Man:
    def __init__(self, name):
        self.name = name
        print("Initialized")
    
    def hello(self):
        print("Hello " + self.name)
    
    def goodbye(self):
        print("Bye " + self.name)

m = Man("David")
m.hello()
m.goodbye()
```

- Man이라는 클래스 정의 → Man 클래스에서 m이라는 인스턴스(객체) 생성.
- Man의 생성자(초기화 메소드)는 name이라는 인수를 받고, 그 인수로 인스턴스 변수인 `self.name`을 초기화.
- 인스턴스 변수(=객체변수): 각 객체별로 저장되는 변수

## ⭐ 넘파이(numpy)

- 배열/행렬 계산을 편리하게 할 수 있도록 도와주는 라이브러리

### 넘파이 가져오기

- 이렇게 가져오면 앞으로 넘파이가 제공하는 메소드를 `np`를 통해 참조 가능

### 넘파이 배열 생성하기

- `np.array()`
    - 넘파이 배열 생성

### 넘파이 산술연산

- 주의점: 배열 x와 배열 y의 원소 수가 같음! ⇒ 원소 수가 다르면 오류가 발생하기 때문에 주의
    - x와 y의 원소 수가 같다면 산술 연산은 각 원소에 대해서 행해짐
    - 원소별 = element-wise
- 넘파이 배열: 원소별 계산 뿐만 아니라 넘파이 배열과 수치 하나(스칼라값)의 조합으로 된 산술연산도 수행 가능.
    - **브로드캐스트**: 스칼라값과의 계산이 넘파이 배열의 원소별로 한 번씩 수행 ⇒ 아마 B(a1+a2) = Ba1+Ba2 (B는 스칼라 벡터 a는 (a1, a2)인 것을 말하는거 같음.

```python
x = np.array([1.0, 2.0, 3.0])
y = np.array([2.0, 4.0, 6.0])

x+y  #원소별 덧셈
>>> array([3., 6., 9.])

x-y #원소별 뺄셈
>>> array([-1., -2., -3.])

x*y #원소별 곱셈
>>> array([2., 8., 18.])

x/y
>>> array([0.5, 0.5, 0.5])
```

### 넘파이 N차원 배열

- 넘파이는 1차원 배열 뿐만 아니라 다차원 배열 역시 작성 가능
- 2차원 배열 작성하는 방법

```python
A = np.array([[1,2], [3,4]])
print(A)

#결과 [[1 2]
#			[3 4]]

A.shape
#결과 (2,2)

A.dtype
#결과 dtype('int64')
```

- 행렬의 형상은 `shape`로 알 수 있고, 행렬에 담긴 원소의 자료형은 `dtype`으로 알 수 있음.
- 서로 형상이 같은 행렬은 산술 연산도 대응하는 원소별로 계산됨
    
    (N차원의 배열에서 배열의 ‘각 차원의 크기(원소 수)’ = ‘배열의 형상’

### 브로드캐스트

- 넘파이에서 형상이 다른 배열끼리도 계산할 수 있음 ⇒ 2x2 행렬에 스칼라 10을 곱할 때, 스칼라 10이라는 값이 2x2의 행렬로 확장된 후 연산 이루어짐

⇒ 일단은 그냥 스칼라 * 벡터(행렬? 배열?)의 곱 느낌으로 받아들임… 그거 표현한 것 같기도..?

```python
A = np.array([[1,2], [3,4]])
B = np.array([10,20])

A*B
#결과 array([[10,40],
#					 [30,80]])
```

- B가 1차원 배열임에도 2차원 배열인 A와 똑같은 형상으로 변형된 후에 원소별 연산이 이루어짐.

<aside>

- ❓ 브로드캐스트가 되면 사실 앞에서 x랑 y의 모양(shape)이 같아야한다고 했는데 아니지 않나?
    
    1. 기본원칙: 모양이 같아야함
    
    기본적으로 요소별로 산술연산이 이루어지기에 짝이 맞아야함
    
    2. 예외조항: 브로드캐스팅
    
    넘파이는 연산하려는 두 배열의 모양이 달라도, 두 가지 조건 중 하나를 만족하면 큰 배열의 모양에 맞게 작은 배열의 모양을 확장시켜 연산을 수행함
    
    **[규칙1] 두 배열의 차원 수가 다르면, 차원이 낮은 배열의 앞(왼쪽)을 1로 채움**
    
    **[규칙2] 각 차원에서 두 배열의 크기가 같거나, 어느 한쪽의 크기가 1이어야함**
    
    예) 
    케이스1: 2차원 & 1차원의 연산
    배열A의 모양 (3,3) 3행 3열 | 배열B의 모양 (3,) 1행 3열처럼 보임
    
    ⇒ B의 모양을 규칙에 따라 (1,3)으로 바꿈. → A의 행(3)과 B의 행(1)을 비교해 1이 있기에 B를 3행으로 복사해 늘림 → 둘 다 (3,3)이 되어 연산 가능!
    
    케이스2: 연산 불가능한 경우
    배열A (3,2) | 배열B (3,)
    - A의 마지막 차원은 **2**인데 B의 마지막 차원은 **3**
    
    ⇒ 둘 다 1이 아니면서 서로 다르기에 넘파이가 늘리지 못해 에러 발생.
    
</aside>

### 원소 접근

- 원소의 인덱스는 0부터 시작

```python
#원소 접근
X = np.array([[51,55], [14,19], [0,4]])
print(X)

print(f"X[0]: {X[0]}") #0행
print(f"X[0][1]: {X[0][1]}") #(0,1) 위치의 원소
```

```python
#원소 접근
X = np.array([[51,55], [14,19], [0,4]])
print(X)

print(f"X[0]: {X[0]}") #0행
print(f"X[0][1]: {X[0][1]}") #(0,1) 위치의 원소
```

- `flatten`: 평탄화 ⇒ 1차원 배열로 변환해줌

```python
#flatten: 평탄화
X = np.array([[51,55], [14,19], [0,4]])
X = X.flatten()

print(X)

#결과: [51 55 14 19  0  4]

#인덱스가 0,2,4인 원소 얻기 (앞에서 1차원 배열로 바꿨으니까)
X[np.array([0,2,4])]

#결과: array([51, 14,  0])
```

- 특정 조건을 만족하는 원소만 얻기

```python
#특정 조건만 만족하는 원소 얻기
print(X > 15) #조건에 대한 각 원소의 T/F 나옴
print(X[X>15])

#결과: [ True  True False  True False False]
#     [51 55 19]
```

## ⭐ 맷플롯립(matplotlib)

- 딥러닝 실험에서는 그래프 그리기와 데이터 시각화 역시 중요함
- 맷플롯립: 그래프를 그려주는 라이브러리로 데이터 시각화가 쉬워짐.

### 단순한 그래프 그리기

- 그래프를 그리기 위해서는 `pyplot` 모듈 이용.

```python
import numpy as np
import matplotlib.pyplot as plt

#데이터 준비
x = np.arange(0,6, 0.1) #0에서 6까지 0.1 간격으로 생성
y = np.sin(x)

#그래프 그리기
plt.plot(x,y)
plt.show()
```

- `arange` 메소드를 [0, 0.1, 0.2 … 5.9] 라는 데이터를 생성해 변수 x에 할당.
- x의 각 원소에 넘파이의 sin 함수인 `np.sin()`을 적용해 변수 y에 할당.
- x와 y를 인수로 `plt.plot` 메소드를 호출해 그래프를 그리고 `plt.show()`를 호출해 그래프를 화면에 출력!

### pyplot의 기능

- cos 함수 그리기 + 제목과 각 축의 이름(레이블) 표시 등 pyplot의 다른 기능 사용하기

```python
from matplotlib.lines import lineStyles
#2. pylot의 기능

#데이터 준비
x = np.arange(0,6, 0.1)
y1 = np.sin(x)
y2 = np.cos(x)

#그래프 그리기
plt.plot(x, y1, label="sin")
plt.plot(x, y2, linestyle="--", label="cos") #cos함수는 점선으로 그리기
plt.xlabel("x") #x축 이름
plt.ylabel("y") #y축 이름
plt.title("sin&cos") #제목

plt.legend()
plt.show()
```

- `plt.plot(x, y2, linestyle="--", label="cos")`
    - `linestyle`: 점선 등 선의 종류 선택. 선의 모양 결정.
    - `label`: 선의 이름표를 붙이는 역할. 나중에 범례(legend) 만들 때 해당 텍스트가 사용됨.
- `plt.xlabel()`, `plt.ylabel()`:
    - 각 축이 무엇을 의미하는지 이름 붙여주기
- `plt.legend()`:
    - `label`로 정해준 이름표들을 모아 그래프 한 구석에 범례(box)로 보여줌.
